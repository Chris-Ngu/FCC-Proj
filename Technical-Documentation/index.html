Introduction

GuestOfFail is a discord bot written primarily for LNG, but will be converted to open-distribution in the future. The bot was created to replace multiple bots and condense the bot lists being taken up in the server as well as writing our own code to perform tasks that no other bot was capable of.

For more information about the project, please refer to the github project repository

Currently being maintained by: Chris Nguyen & Jonah Bui

Commands
=ping
  • Check the server ping to the bot
    • =clear #
  • Clears a certain amount of messages (# = input, ROLE RESTRICTED: Garbage Man)
    • =contact
  • Shows the contact/ suggestion form for input + changes to the bot
    • =invite
  • Creates an invite link for the server, will send a message to the owner and in chat channel
    • =kick @MEMBER
  • Kicks a member from the discord server (@MEMBER = mention, ROLE RESTRICTED: Garbage Man)
    • =mute @MEMBER REASON
  • Mutes a member from a discord voice channel (@MEMBER = mention; REASON= sentence, ROLE RESTRICTED: Garbage Man)
    • Help
    =help
  • Displays the help function into the server with more information
    • =info
  • Displays the server information/ statistics

    • If you need any help with the documentation, configuration, or even setting up contact the contributing members in the introduction
